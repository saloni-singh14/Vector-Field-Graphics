This function plots the pixels (x,y) in 8 different symmetric segments of the circle. For whole 360 degree of circle we will divide it in 8-parts each octant of 45 degree. For each pixel (x,y) we plot the following coordinates:

(x,y)

(x,-y)

(-x,y)

(-x,-y)

(y,x)

(y,-x)

(-y,x)

(-y,-x)
